# Healthcare Claims Dashboard

## Overview

This project contains a Power BI dashboard focused on Healthcare Claims Analysis. The dashboard is designed to help analyze claim performance, payment trends, claim categories, and operational metrics using interactive visualizations.

The report enables users to:

* Monitor healthcare claim costs and payments
* Identify high-cost claim categories
* Compare billed vs paid amounts
* Analyze claim distribution patterns
* Track claim performance through KPIs and charts

---

## File Information

* **Project File:** `Healthcare_Claim.pbix`
* **Tool Used:** Microsoft Power BI
* **Dashboard Type:** Healthcare Claims Analytics

---

## Dashboard Features

The Power BI dashboard includes the following visualizations:

### KPI & Summary Metrics

* Card visual for key healthcare claim metrics
* Gauge chart for performance tracking

### Charts & Analysis

* Clustered Column Charts

  * Claim cost comparison
  * Paid vs billed amount analysis

* Pie Chart

  * Claim category distribution

* Treemap

  * High-cost claim segmentation

* Scatter Charts

  * Relationship analysis between claim metrics
  * Cost and payment trend visualization

### Interactive Components

* Advanced slicer for filtering data dynamically
* Interactive visuals for drill-down analysis
